LED mirror is en interactive led installation made by Jeroen van Goor and Johan ten Broeke of Fullscreen.nl. It currently runs a few different interactive applications. For instance loop recording app, a 'painting with highlights' app and an app displaying silhouette of spectators using a difference matte. (showcased in the video on the home page)

All hard- and software design is done by Fullscreen.nl and makes use of a Raspberry Pi with a Raspicam camera module. If you are interested in the technical details, like to read source code or build your own LED mirror. The project is on github
